# Loop Self-Serve Onboarding Mock UI

An interactive mock UI demonstrating the redesigned Loop self-serve onboarding flow.

## Overview
Built to reference the team redesign of the onboarding process from Closed Won → 3PD platform access verified.

## Features
- 5-screen interactive flow (Welcome → Platforms → Stores → Products → Go Live)
- 3-column parallel platform connection view (DoorDash | Uber Eats | GrubHub)
- Simulated access verification with error states and reverify flows
- Loop Assistant buddy chat
- Toast notifications
- Matches Loop design system (Sora font, Loop navy/teal palette)

## How to use
Open `Loop_Onboarding_Mock.html` directly in any browser — no build step required.
